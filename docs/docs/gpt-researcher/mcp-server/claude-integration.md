# Claude-Integration

Mit der Claude-Integration kannst du den GPT-Researcher-MCP-Server mit Claude API oder Claude Desktop verbinden.

## API-Integration

Verwende diese Variante, wenn du Claude programmgesteuert an den MCP-Server anbinden willst.

## Desktop-Integration

Verwende diese Variante, wenn du Claude Desktop auf dem Mac einsetzen möchtest.

## Vorgehen

1. MCP-Server starten
2. Claude so konfigurieren, dass es auf den Server zeigt
3. Die Tools `quick_search`, `deep_research` und `write_report` testen

## Hinweise

- Stelle sicher, dass deine API-Keys korrekt gesetzt sind
- Prüfe die Server-URL, falls keine Antworten zurückkommen
- Für aufwendige Aufgaben ist `deep_research` meist die bessere Wahl
