# Vanilla-JS-Frontend

Das VanillaJS-Frontend ist eine leichtgewichtige Lösung, die FastAPI nutzt, um statische Dateien auszuliefern.

#### Voraussetzungen
- Python 3.11+
- pip

#### Einrichtung und Start

1. Benötigte Pakete installieren:
   ```bash
   pip install -r requirements.txt
   ```

2. Server starten:
   ```bash
   python -m uvicorn main:app
   ```

3. Unter `http://localhost:8000` öffnen
